**Project:** Comparison of Rule-Based AI (ELIZA) and Modern Generative AI (LLM)

## Overview
This laboratory project explores the evolution of Artificial Intelligence. It compares a historical, rule-based chatbot (ELIZA) using the NLTK library with a modern Large Language Model (Qwen2.5-1.5B-Instruct).

## Files in this Repository
* `eliza.py`: A rule-based chatbot with 5 modified response patterns.
* `LLM.py`: A generative AI script utilizing modern LLM logic.
* `chat_comparison.py`: A Tkinter-based GUI that displays both AI models side-by-side for direct comparison.

## How to Run
1. Ensure you have Python installed.
2. Install dependencies: `pip install nltk transformers`
3. Run the comparison GUI: `python chat_comparison.py`
